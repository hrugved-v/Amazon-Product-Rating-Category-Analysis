# 📊 Product Rating & Category Analysis

## 📌 Overview
This project focuses on analyzing product data to uncover insights related to ratings and category performance. The goal is to understand patterns in customer preferences using data analysis techniques.

## 🎯 Objectives
- Clean and preprocess raw data  
- Perform Exploratory Data Analysis (EDA)  
- Visualize trends and patterns  
- Extract meaningful insights  

## 📂 Dataset
The dataset contains product-related information such as categories, ratings, and other attributes used for analysis.

## 🛠 Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

## 🔍 Analysis Performed
- Data Cleaning (handling missing values, duplicates)  
- Univariate and Bivariate Analysis  
- Correlation Analysis  
- Data Visualization  

## 📊 Key Insights
- Certain product categories have higher average ratings  
- Variations exist across different categories  
- Data trends indicate customer preferences
